# Windows Server Active Directory Lab

## Overview
This project documents a Windows Server lab environment built to practice enterprise systems administration, Active Directory administration, DNS, DHCP, Group Policy, user/group management, and access control.

## Technologies Used
- Windows Server 2019/2022
- Active Directory Domain Services
- DNS
- DHCP
- Group Policy
- PowerShell
- Hyper-V
- Windows client systems

## What I Configured
- Installed Active Directory Domain Services
- Created and validated a domain environment
- Configured DNS and reverse lookup zones
- Configured DHCP services
- Created organizational units, users, and groups
- Managed users and groups using GUI tools and PowerShell
- Joined client systems to the domain
- Configured shared folders and access permissions
- Validated replication and domain health using administrative tools

## Skills Demonstrated
- Windows Server administration
- Active Directory administration
- DNS/DHCP configuration
- Group Policy management
- User and group administration
- PowerShell administration
- Access control and documentation
- Enterprise troubleshooting
